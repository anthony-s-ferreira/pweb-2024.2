Explique o papel do JavaScript no desenvolvimento web moderno.
Por que ele é considerado uma linguagem essencial para a
criação de sites interativos?

Tanto para o desenvolvimento Frontend como Backend, JavaScript é muito utilizado 
no desenvolvimento web moderno, como uma linguagem simples e que responde à ação do 
usuário, sendo multiparadigma e, mais importante, interpretada no navegador.
A necessidade de criar sites dinâmicos cresceu rapidamente na década de 1990. Em 
1995, Brendan Eich desenvolveu o JavaScript para atender a essa demanda, tornando 
os sites mais dinâmicos e interativos. Com JavaScript, é possível manipular elementos 
da página em tempo real, atualizar conteúdos dinamicamente e criar interfaces modernas 
e responsivas, características essenciais para a web atual.
